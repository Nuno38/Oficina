# Sistema de uma Oficina

° Clientes e veículos
<br>
° Ofcina e Time mecânico
<br>
° Tipo de serviço prestado (" Manutenção ou Revisão")
<br>
° Ordem de serviços
<br>
° Gerenciamento de peças e estoque
<br>
° Verificação de status do serviço
